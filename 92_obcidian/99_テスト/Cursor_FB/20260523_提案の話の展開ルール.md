# Cursor FB — 提案の話の展開ルール

## 内容

何か提案を行う場合は、次のルールを守る。

1. **現状 → 課題 → ソリューション** の順で話を展開する
2. **現状**は、全体構成 → 今回話したい部分の説明 → そこの現状
3. **課題**も、全課題の MECE な洗い出し → 今回特に話したい部分の特定
